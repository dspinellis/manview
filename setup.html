Manview — Unix manual online viewer

Manview dynamically presents a Unix manual page formatted in troff  as an HTML web page. Through this form you can create a URL for any Unix manual page hosted on GitHub. The URL will display the page as nicely formatted HTML. (See this example.) This allows you to link to automatically-formatted human-readable Unix tool documentation, saving you the trouble of having to format it yourself and the worry of keeping the formatted version current.


To obtain the URL, click on the Raw button when viewing a GitHub project's manual page file content and copy the displayed page's link. (Don't use the Raw button's hyperlink, as this redirects to another domain.)
